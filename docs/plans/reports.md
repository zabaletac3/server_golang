## Idea
Implementar el módulo de reportes en 3 fases: reportes básicos, inventario, y dashboard avanzado

## Implementation
- Fase 1a: Crear schema.go con Report
- Fase 1b: Crear repository.go con aggregations
- Fase 1c: Crear service.go y handler.go
- Fase 2a: Reporte de citas (por fecha, estado, veterinario)
- Fase 2b: Reporte de ventas (ingresos, métodos de pago)
- Fase 2c: Reporte de pacientes
- Fase 3a: Reportes de inventario (stock bajo, expiración)
- Fase 3b: Dashboard con métricas
- Fase 3c: Exportar a PDF/Excel
